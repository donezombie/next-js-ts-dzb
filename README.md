## Packages use in template:

- Typescript
- React.js
- Next.js
- SCSS
- Redux
- Redux Saga
- Axios
- Formik
- Yup Validator
- i18n (multi lang)

## How to use

```javascript
npm i
npm run dev ( development )

npm run build
npm run publish (build & export)
```
